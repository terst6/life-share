  <h2>从遗传学与人类进化角度保持生活方式的延续性</h2>
  <p>这些改变不是为了让我们像机器一样精准地活着，而是为了避免傻傻地活着，却违背大量的基本原理与准则，进入对健康有害的误区。

  </p><p>这是我个人生活知识的系统性总结，记录了从底层原理到生活改变的过程。请根据自己的理解与 AI 多沟通讨论，去不断更新自己的认知与判断，跟上科技的进步。

  </p><p>从人类进化过程中的饮食习惯与生活方式积累，构筑了我们身体器官以及全身菌群的协同生活模式，过快的现代化进程和廉价生活成本，让我们的身体能够快速获取超出处理极限的营养摄入。导致身体器官的全身营养中毒，这可能是糖尿病、痛风、肝硬化等疾病的病因。世界在飞速工业化现代化，但我们的身体进化速度仍停留在农耕时代，尚未适应现代社会的快速变化。，请放慢食物营养的摄入速度，给身体进化提供一个缓冲时时间。
  </p><p>现代的工业化产品，都是以利益在驱动，盲目相信这些短时间内发明的东西，可以代替人类几百万年进化出来的能力，可能更容易摧毁健康

  </p><p>比如在猴时代，我们吃鸟蛋水果，所以这些东西肯定适合我们的身体，只要不过度摄取；比如我们有百万年吃脂肪的历史，我会吃动物脂肪；比如10万年前人类大量摄入脂肪带来的脑容量4倍增加，让我怀疑如果不保持足够的脂肪摄入，可能会导致脑萎缩风险，这也是我不敢完全素食的原因。比如我发现抽油烟机中的油像橡胶一样难以清理，让我想到这些橡胶化的东西通过鼻腔通道在大脑累积，所以我不再吃常规菜油，改用少量动物油水煮代替。额外尝试一些橄榄油及亚麻籽油

  </p><p>我不相信传统的感冒治疗逻辑，因为我们可以与病毒保持一定距离并靠身体与病毒的共存能力。不相信眼镜行业的金标准，去佩戴眼镜，因为我相信自己的眼镜可以调节回去。不相信补钙逻辑，我认为缺钙只是因为缺乏足够的运动或骨骼负荷不足，导致身体排斥对钙的吸收。

  </p><p>根据提前预防与准备，构筑身体的健康防线，把自己当成糖尿病患者一样控制饮食，预防糖尿病痛风，把自己当癌症病人一样去饮食，尽量让身体处于抗癌环境中，减少癌细胞的生成。比如提高免疫力的各种生活方式改变。

  </p><p>另外，塑料颗粒在大脑中的风险及预防，也需要在生活的方方面面进行调整与控制。

  </p><p>在这些认知状况下，最伤害健康并摧毁身体的生活方式之一就是：将所有营养脂肪放入锅中，长时间炖煮食物，让营养物质可以大量快速被身体吸收。比如让每一炖饭都是高营养一锅炖出来吃，并保持好吃的口味与盐分，长时间这样的饮食方式，可能在几个月内对身体产生不良影响。容易被身体吸收与水解，是评断食物与身体友好程度的一个标准，但是不能经常大量的吃，让食物至少有超过60%都处于快速加工状态，模拟上一代人没有电饭煲、缺乏时间煮饭的生活节奏，给肾脏肝脏慢速处理食物的机会，特别是晚餐不要留给身体太多负担
